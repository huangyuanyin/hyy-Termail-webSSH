# hyy-webSSH-Termail
基于xterm3 + websocket + python 实现网页版在线终端通过SSH连接服务器

## 运行步骤
1.前端部分

```
cd webssh 
npm install 
npm run dev 
```

2.后端部分
```
pip install -r -requirement.txt
python .\server.py
```
